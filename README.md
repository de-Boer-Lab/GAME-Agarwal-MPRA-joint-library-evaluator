# GAME Agarwal MPRA Joint Library Evaluator

Agarwal et al. (2025) MPRA evaluator first requests log-scale predictions for ~56k probes across WTC11, K562, and HepG2 cells. It then calculates Pearson correlations between predicted and measured $(\log_2(rna/dna))$. It also assesses cell-type specificity by correlating predicted vs. observed expression differences.
